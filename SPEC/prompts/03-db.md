STEP 3 — Storage DB (Mongo)

Aggiungi MongoDB come storage.
- docker compose userà mongo
- salva in collections:
  - reports_raw (metadata + raw JSON)
  - reports_normalized (campi utili per dashboard)

Definisci schema minimo normalized per:
- report_type
- report_start_day
- report_end_day
- part_index
- ingested_at
- payload (o subset)

Alla fine:
- script di ingest che salva su mongo
- comando di verifica (es. endpoint o query)
